## Log into server with SSH keys


### Generate SSH keys


```shell
ssh-keygen -t rsa
```


Your keys will be created at _~/.ssh/id_rsa.pub_ and _~/.ssh/id_rsa_


### Transfer Your Public Key to the Server


```shell
ssh-copy-id root@server-ip
```
